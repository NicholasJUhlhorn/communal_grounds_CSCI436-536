Introducing Communal Grounds

A project for CSCI436/536: Technology for Social Good with Dr. Shameem Ahmed.

Communal Grounds is a volunteer organization web application built with Flask. Its core functionality focuses on managing community projects and the people that work on them.

Key Features:

  • User Management: The platform tracks user state, allows sign-up with passwords, and enables current users to edit their profiles and manage a friend list.
  • Project Organization: Projects are displayed on the home page and users can apply to join a project, allowing hosts to handpick members for better security and fit.
  • Detailed Profiles: User profiles show active projects and participation percentages to promote active participation.
  • Project Communication & Delegation: Each project has a central hub. Project owners can assign specific roles or job titles to users to clearly define responsibilities and tasks for effective work delegation.

Technologies Used

The Communal Grounds application is built on a Model-View-Controller (MVC) architectural pattern.

  • Database: The application uses a SQLite DB to store all platform data, including tables for Users, Projects, Project Members, and Reactions.
  • Backend Framework: Python and a framework like Flask are used for the Controller and Model layers. This includes logic for handling user input, managing session state, and interacting with the database.
  • Web Technology: Standard HTML, CSS (for style and feel), and dynamic routing are used for the View and presentation layer.
